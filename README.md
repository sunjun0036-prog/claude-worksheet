# claude-worksheet

A small practice project used to learn the GitHub PR workflow.

## What's here

- `src/math.js` — a couple of simple math helper functions.

## Usage

```js
const { add, subtract } = require('./src/math');

add(2, 3); // 5
subtract(5, 2); // 3
```

This repo is intentionally minimal so its history is easy to follow.
