# 수주통보 게시판 새 글 알림 프로그램

사내 게시판(수주통보)에 새 글이 올라오면 **메일로 알려주고, 수주통보서/Quotation PDF를
자동으로 받아 첨부까지 해주는** 프로그램입니다.
메일은 SMTP로 직접 발송하므로 Outlook 보안 팝업이 뜨지 않습니다.

## 사용법 (평소엔 이것만)

### `2_감시시작.bat` 더블클릭
1. 검은 콘솔 창 + 크롬 창이 뜹니다.
2. 크롬에 로그인 화면이 보이면 **사내 계정으로 로그인**하세요. (로그인하면 자동으로 감시 시작)
3. 로그인 후 **크롬 창은 닫지 말고 최소화**만 하세요. (닫으면 로그인이 풀립니다)
4. 콘솔에 1분마다 `확인 완료 — 새 글 없음` 이 찍히면 정상 동작 중입니다.
5. 감시를 멈추려면 검은 콘솔 창을 닫으면 됩니다.

### 새 글이 올라오면 자동으로
1. 글을 열어 첨부 중 파일명에 **`수주통보서` 또는 `Quotation`**(대소문자 무관)이
   들어간 파일을 다운로드
2. 글 제목의 수주통보 번호로 하위 폴더를 만들어 저장 —
   예: `수주통보서_다운로드\HMT26-142\` 에 수주통보서+Quotation PDF가 함께 들어감
   (Rev.1 개정본도 같은 번호 폴더에 저장되어 원본과 모임, 원본 한글 이름 유지)
3. 알림 메일 발송 — 제목/작성자/등록일시/글번호 + 위 PDF들 첨부
   (메일 첨부 이름은 게이트웨이의 한글 파일명 제거 문제 때문에 영문으로 변환됨:
   `수주통보서(HMT26-142).pdf` → `OrderNotice(HMT26-142).pdf`)

## 보조 도구 (문제 있을 때만)
- `3_메일테스트.bat` : SMTP 메일 발송만 테스트
- `4_진단.bat`      : 크롬을 띄워 로그인 상태/목록 API 를 눈으로 확인
- `5_첨부탐색.bat`  : 게시글 첨부 영역의 HTML 구조 덤프 (개발용)
- `6_첨부테스트.bat`: 특정 글(기본 107615)에서 첨부 다운로드→메일 첨부 전체 흐름 테스트
- `1_로그인.bat`    : (현재는 불필요 — 로그인은 감시 창에서 바로 하면 됨)

## 설정 바꾸기 (order_watcher.py 위쪽)
- `MAIL_TO`        : 알림 받을 메일 주소
- `POLL_SECONDS`   : 확인 주기(초). 기본 60초.
- `TITLE_FILTER`   : 특정 단어가 제목에 있는 글만 알림 (비우면 전체)
- `ATTACH_ENABLE`  : 첨부 자동 다운로드/첨부 기능 켜기(True)/끄기(False)
- `ATTACH_FILTERS` : 다운로드할 첨부 파일명 키워드 목록 (기본: 수주통보서, Quotation)
- `FILENAME_MAP`   : 메일 첨부 시 한글→영문 이름 변환 규칙

## 메일(SMTP) 설정
- 기본: 회사 게이트웨이 `wblock.hanbalmasstech.com:25`, 인증 불필요.
- 개인 Gmail 로 바꾸려면 order_watcher.py 의 SMTP_* 주석 참고
  (smtp.gmail.com:587 + 앱 비밀번호).

## 로그인이 풀리면
- "로그인 필요" 메일이 한 번 오고, 콘솔에도 표시됩니다.
- **감시 크롬 창에서 다시 로그인만 하면** 자동으로 감시가 재개됩니다. (재시작 불필요)

## PC 켤 때 자동 시작
`2_감시시작.bat` 바로가기를 `Win+R` → `shell:startup` 폴더에 넣으세요.
(단, 부팅 후 크롬 창에서 한 번 로그인해줘야 합니다)

## 파일들
- `order_watcher.log`    : 실행 기록
- `state.json`           : 마지막으로 확인한 글번호 (지우면 기준선 다시 설정)
- `browser_profile/`     : 브라우저 프로필
- `수주통보서_다운로드/` : 다운로드된 PDF 저장 폴더
